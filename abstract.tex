{\Large \textbf{ABSTRACT}}
\end{center}
\justifying
\noindent
ProblySUS is a comprehensive web-based security system designed to detect scam websites, phishing pages, and fraudulent online platforms using a multi-signal analysis approach. This project addresses the limitations of traditional detection methods by combining multiple independent indicators such as domain intelligence, DNS analysis, threat intelligence feeds, content inspection, and real-time browser behavior analysis to provide a reliable and explainable risk assessment of websites.
\noindent
The system integrates advanced techniques including headless browser simulation using Playwright to capture dynamic website behavior such as redirects and external network requests, which are not detectable through static analysis alone. Additionally, ProblySUS incorporates privacy auditing and tracker identification to evaluate website data collection practices. Built using Python and Flask for the backend, and React with Tailwind CSS for the frontend, the platform also includes a browser extension for real-time URL scanning within the user's browsing environment.
\noindent
The system generates a composite risk score ranging from 0 to 100 and classifies websites into Safe, Caution, Suspicious, or Fraudulent categories, along with clear explanations for each decision. With features such as parallel processing, local threat feed integration, and a fast analysis mode, ProblySUS provides a scalable and efficient solution for enhancing user awareness and improving online safety.
\noindent
\textbf{Keywords:} Scam Detection, Phishing Detection, Website Security, Threat Intelligence, DNS Analysis, Headless Browser, Privacy Auditing, Risk Scoring, Web Application, Browser Extension
